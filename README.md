# RiverD Xxf86vm
Dynamic and static D bindings for X11 XFree86 video mode extension library

## Usage
```d
import riverd.xxf86vm;

void main()
{
	// Load the Xxf86vm library.
	dylib_load_xxf86vm();

	// Now Xxf86vm functions can be called.
	...
}
```
